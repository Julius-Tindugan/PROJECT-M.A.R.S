# Setup Quick Reference

## Free Tier Services Used

### 1. **Netlify** (Frontend Hosting)
- Free up to 100 GB bandwidth/month
- Automatic HTTPS
- Deploy from git on pushes

### 2. **Render** (Backend Hosting)
- Free tier ($0/month)
- PostgreSQL databases free
- GitHub Actions integration
- *Note: Spins down after 15 min inactivity*

### 3. **Neon** (Database)
- Free PostgreSQL tier
- 3 projects, 512 MB RAM
- Auto-backups included

### 4. **GitHub Actions** (CI/CD)
- 2000 free minutes per month
- Runs tests automatically

## Quick Deployment Checklist

- [ ] Push code to GitHub `main` branch
- [ ] Create Netlify account, connect GitHub repo
- [ ] Create Render account, deploy backend service
- [ ] Create Neon account, set up PostgreSQL database
- [ ] Add all environment variables to Netlify
- [ ] Add all secrets to GitHub (NETLIFY_AUTH_TOKEN, NETLIFY_SITE_ID, RENDER_DEPLOY_HOOK)
- [ ] Test frontend URL in browser
- [ ] Test backend API endpoints with curl
- [ ] Monitor GitHub Actions for any failures

## Environment Variables Template

### For Netlify (Frontend)
```
VITE_API_URL=https://your-backend.onrender.com
```

### For Render (Backend)
```
APP_NAME=Your App Name
APP_ENV=production
APP_DEBUG=false
APP_URL=https://your-backend.onrender.com
APP_KEY=[generated by Laravel]
DB_CONNECTION=pgsql
DB_HOST=your.neon.tech
DB_PORT=5432
DB_DATABASE=mydb
DB_USERNAME=myuser
DB_PASSWORD=mypassword
CACHE_DRIVER=database
SESSION_DRIVER=cookie
QUEUE_CONNECTION=sync
```

## Why This Architecture?

- **Netlify**: No-config React deployment, includes Express/SSR
- **Render**: Excellent free tier, integrates with GitHub
- **Neon**: Most generous free PostgreSQL database
- **GitHub Actions**: Seamless integration, automatically triggers

This setup requires ZERO dollars while you're in development/testing phase!
